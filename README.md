# Flask project

1 Install venv:

    $ apt-get install python3-venv

2 Create virtualenv

    $ python3 -m venv venv

3 Activate the venv

    $ . venv/bin/activate

4 Install flask

    $ pip install Flask

5 Export Flask variable

    $ export FLASK_APP=application.py

6  Install and run flask

    $ flask run

7 Install Requirements

    $ pip install -r requirements.txt
